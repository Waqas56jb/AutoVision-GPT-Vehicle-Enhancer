# Preset backgrounds

Any image file (`.png`, `.jpg`, `.jpeg`, `.webp`) placed in this folder automatically
appears as a selectable **Saved background** in the app — no code changes needed.

The filename becomes the display name (e.g. `warm-showroom.png` → "Warm Showroom").

Drop your dealership's approved background scenes here to replace or add to the
starter backdrops shipped with the project.
